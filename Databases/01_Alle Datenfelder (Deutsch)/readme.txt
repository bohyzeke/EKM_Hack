
Beschreibung der Demo-Datenbank 'Alle Datenfelder':

Diese Demo-Datenbank zeigt die Möglichkeiten von mehreren Benutzern, die unterschiedliche Daten sehen können. Jeder verfügbare Datentyp wird in dieser Datenbank ein mal verwendet, um die verschiedenen Anwendungsbereiche aufzuzeigen. Die Benutzer 'ekm' und 'admin' sehen alle Daten. Der Benutzer 'db' sieht nur die Daten aus der PC-Datenbank, der Benutzer 'Key' sieht nur die Daten auf dem Electronic-Key. Da sich alle Datenelemente auf jeweils einer 'Group-Box' befinden, wird nur die entsprechende 'Group-Box' über das Benutzerrecht sichtbar gemacht.


Die installierten Benutzer sind:

Benutzer                Passwort

admin                   admin
ekm                     ekm
db                   	db
key                     key
